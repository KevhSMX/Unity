# Unity
### Projecte del videojoc en Unity:
Per tal de dur a terme la pràctica, caldrà anar dissenyant el joc en aquest ordre:

1. Crear el personatge principal i programar el seu moviment
(dreta/esquerra) i el doble salt. La càmera l’haurà de seguir. El
comptador de salts es torna a posar a 0 si es toca el terra o una
plataforma.

2. Crear l’escenari amb l’eina que es consideri més adient o descarregant-
lo d’Internet. Ha de contenir almenys 3 plataformes.

3. Crear dos enemics amb diferents característiques: uns es podran matar i
altres els haurem d’esquivar. Els primers hauran d’aparèixer de manera
aleatòria al cap d’amunt de l’escenari i caure, són els que no es poden
matar. Els segons poden aparèixer de posicions fixes, han de seguir al
jugador i es podran matar amb l’arma). Tots dos han d’estar ja amb em
el seu sprite corresponent.

4. Crear l’arma que tindrà el personatge per matar els enemics (es pot triar
l’arma que es vulgui).

5. El personatge principal ha de tenir 3 vides i les ha de perdre si toca els
enemics.

6. Crear una UI de joc. Com a mínim, ha de mostrar les vides restants i el
temps que es porta de joc.

7. Crear una pantalla d’inici amb un botó per començar a jugar i una
pantalla de recàrrega del joc.

#### OPCIONAL:
• Crear animacions per al personatge principal o enemics.
• Mostrar màximes puntuacions a la pantalla de recàrrega.
• Altres millores
